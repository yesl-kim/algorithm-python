## 파일 읽기

### input

- input()은 한 줄씩 읽어들인다.
- ex. input() -> 파일의 1번 줄 -> input() -> 파일의 2번 줄

### readlines

- 파일의 한 줄씩 읽는 코드
- 줄바꿈 코드도 함께 삽입됨

### read

- 파일 전체를 읽음
- 개인적으로 파일 전체를 읽고 줄바꿈을 기준으로 split 하는게 값을 읽어들이는 것이 편했음
- 그런데 가장 편한건 input...이더라

## format, %

### %

- %* => *자리를 원하는 값으로 바꿔줌
- \_자리에는 대체할 값의 형식을 넣어준다 (d, s, f)
- d: 정수, s: 문자열, f: 실수

```py
print("#%d %d", %(1, 2)) # => #1 2
```
